# Xbox Sales Dashboard

## Objective
The goal of this challenge is to create a **sales dashboard**, focusing on data organization and visualization. The purpose is to transform raw data into clear and useful visual information, enabling effective sales performance analysis and data-driven decision-making.

## Data Source
- The data is stored in an **Excel file**;
- The dashboard was developed directly in Excel;
- Execution is simple: just open the file and use it.

## Context
The information represents **Xbox subscription sales** during the year **2024**, covering the following subscription types:
- Monthly;
- Annual;
- 4-month.

## How to Run
1. Download or clone this repository;
2. Open the provided Excel file;
3. Explore.

## Notes
- No need to install additional libraries or software;
- All work was done within Excel.

This project is a practical exercise in **[data visualization](ca://s?q=Data_visualization)** and **[sales analysis(ca://s?q=Sales_analysis)**, showing how simple data can be transformed into valuable insights.
